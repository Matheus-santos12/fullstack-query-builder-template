# Query Builder

Projeto desenvolvido durante as aulas da Rocketseat para praticar a construção de queries com Knex.js em uma API Node.js com TypeScript.

## Tecnologias

- Node.js
- TypeScript
- Express
- Knex.js
- SQLite

## Como rodar o projeto

**Pré-requisitos:** Node.js instalado na sua máquina.

1. Clone o repositório:

```bash
git clone https://github.com/Matheus-santos12/fullstack-query-builder-template.git
```

2. Instale as dependências:

```bash
npm install
```

3. Rode as migrations:

```bash
npm run knex -- migrate:latest
```

4. Inicie o servidor:

```bash
npm run dev
```

## Aprendizados

- Como usar o Knex.js como query builder
- Criação e uso de migrations para gerenciar o banco de dados
- Construção de uma API REST com Express e TypeScript
